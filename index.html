Jan 2019 - Apr 2019

Fingerprint Based Biometric Voting Machine.

  • ACADEMIC PROJECT

It uses the Fingerprint technology, Arduino System and allows users to submit Fingerprints at the polling booth. It reads the data from the Fingerprint and verifies the data which is already stored in the database. If the given data matches with the database data, then the system allows the person to cast their vote. Otherwise, it immediately activates the display and alerts the security authorities.

May 2019

Predicting The Readmission To Hospitals For Diabetic Patients.

  • INTERNSHIP PROJECT

By utilising a model created with Python and Node-RED and following the steps of data exploration, feature engineering, building training/validation/test samples, model selection, and evaluation, this app aids in the prediction of readmission. Users can enter information about any new patient along with their patient data into the database, and the model will determine whether a patient has to be readmitted within 30 days or not.

Dec 2019 – Jun 2020

Road Lane Detection for Autonomous Vehicles.

  • ACADEMIC PROJECT

It makes use of OpenCV, a computer vision algorithm, and features like the Canny Function that enable edge detection. A zero-intensity mask is prepared, the region of interest is mapped using bitwise operations, the Hough Transform technique is used to find straight lines in the image, polar coordinates are used to determine the slope of vertical and horizontal lines, and the lane lines are identified by combining the lane image with the zero-intensity image.

May 2019

Security Lock with Intrusion Detection and Facial Recognition.

  • INTERNSHIP PROJECT

It is a intrusion detection system built on model that is based on facial recognition, the solution is based on a set of authorised people, it exploits a facial recognition algorithm to determine if the person entering the monitored environment belongs to the set of authorised or not.
